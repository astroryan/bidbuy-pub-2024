환불 신청 안내

  1. 환불 처리 소요시간은 정상영업시간 기준 24시간이내에 환불처리가 완료됩니다.
  2. 2개월 이내의 카드결제시 카드결제취소처리가 됩니다.
    (카드사 반영은 약 7일정도 시간이 소요되십니다) 가상계좌/온라인입금시 은행 계좌로 환불처리 됩니다.
  • 예치금 환불내역은 예치금충전 메뉴에서 예치금 거래내역을 통해 확인 가능합니다.
  • 보증금 환불내역은 보증금 메뉴에서 보증금 거래내역을 통해 확인 가능합니다.
  • 보증금 환불신청을 하시면 해당국가의 관심물품이 모두 삭제됩니다.
